App.java contains main.

===To compile and run (Linux)===
1) Click on green "Clone or download"
2) Download ZIP
3) Extract ZIP
4) In terminal, change directory to src directory in newly extracted folder.
   Path should be Introductory-Programming-Exercises-master/src
5) Compile Java files with the following command:
   javac *.java
   Java compiler should create .class files in the same directory
6) Run the program with the following command:
   java App
